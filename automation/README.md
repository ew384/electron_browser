# 微信视频号自动化发布系统

## 🚀 快速开始

### 1. 安装依赖
```bash
cd automation
npm install
```

### 2. 生成配置文件
```bash
npm run init
```

### 3. 运行测试
```bash
npm test
```

### 4. 启动服务器
```bash
npm start
```

## 📋 功能特性

- ✅ 视频、图文、音乐、音频四种工作流
- ✅ 批量发布到多个账号
- ✅ 模板引擎支持
- ✅ REST API接口
- ✅ CLI命令行工具

## 🎯 使用示例

```bash
# 生成配置模板
node cli/automation-cli.js init --type video

# 运行测试
node cli/automation-cli.js test

# 启动API服务器
node cli/automation-cli.js server --port 3001
```
